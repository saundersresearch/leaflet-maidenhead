# Changelog 

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.2] - 2026-01-19

### Changed
- Changed grid highlighting logic to highlight a grid if either its parent or any of its children are to be highlighted
- Added configuration options to change zoom levels and font sizes

### Removed
- Removed title_size option for font size configuration and replaced with calculation based on zoom level

## [0.2.1] - 2026-01-19

### Fixed
- Fixed centering of labels in grid squares

## [0.2.0] - 2026-01-19

### Added 
- Added CHANGELOG.md
- Added option to highlight specific grid squares
- Added font size configuration option

### Changed
- Refactored project structure using vite
- Centered labels in grid squares
- Updated README.md with usage and installation instructions
